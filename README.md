# WhatsApp-Clone-MERN

This app functions similarly to the popular communication application, WhatsApp.   

Technologies Used:

- MongoDB
- Express
- React
- Node
- Mongoose
- Pushit (to make our mongo database real time)
- Firebase (to host front end)
- Heroku (to host back end)
