# WhatsApp-Clone-MERN

This app functions similarly to the popular communication application, WhatsApp.   

Technologies Used:

- MongoDB
- Express
- React
- Node
- Mongoose
- Pusher (to make our mongo database real time)
- Firebase (frontend)
    - authentication
    - hosting
- Heroku (backend)
